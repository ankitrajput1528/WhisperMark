# WhisperMark

Welcome to WhisperMark, a bookmark website project designed to help users save, organize, and share their favorite web links seamlessly.

## Features
- **Save and Organize Links**: Save your favorite links in categorized folders.
- **Share Bookmarks**: Easily share your bookmarked links with friends or colleagues.
- **Tagging**: Use tags to quickly find related bookmarks.
- **Search Functionality**: Quickly search through your bookmarks.
- **Responsive Design**: Access your bookmarks from any device.

## Getting Started
To get started with WhisperMark:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/ankitrajput1528/WhisperMark.git
   cd WhisperMark
   ```
   
2. **Install Dependencies**:
   ```bash
   npm install
   ```
   
3. **Run the Project**:
   ```bash
   npm start
   ```
   Visit `http://localhost:3000` in your browser.
